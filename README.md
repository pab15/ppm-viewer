# PPM-Viewer
GUI editor for PPM Images. Built in C++ using FLTK. Includes 14 Image Manipulations and Saving Functionality:
1. Remove Red
2. Remove Green
3. Remove Blue
4. Negate Red
5. Negate Green
6. Negate Blue
7. Greyscale
8. RandomNoise
9. HighContrast
10. HorizontalFlip
11.	VerticalFlip
12.	PixelateImage
13.	BlurImage
14.	Rotate90
